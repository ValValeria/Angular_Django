Python urls:

- GET api/carousel/:type 
The structure of json response:
{
data: {
   images: ["/assets/img.png"]
 }
}

- POST /api/carousel/download/:type
The structure of body's request (formdata)
  * type - filename - content
  Also there is the file, named `urls_list.json`, which contains the url of each image

- DELETE /api/carousel/delete/:id
